# 什么值得买

这里，我想罗列一个清单，我们动用集合知的力量来形成一个Minimalist的用品清单，详细规定我们所需要的所有物品。也希望各位积极贡献内容。

该清单应当能够指导我们的购物。我们可能需要清单上的物品，而清单以外的物品我们可能并不需要。各位可以根据自己的经验来扩充、修改该清单。
